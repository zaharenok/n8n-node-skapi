## [1.0.0] - 2024-06-29

### Added
- Initial release of SkAPI.pro n8n node
- Join Request operations:
  - Check pending join requests
  - Process join requests (approve/decline)
- Message operations:
  - Check new messages in groups
- Notification operations:
  - Check Skool notifications
- JWT authentication support
- Client ID tracking
- Full TypeScript implementation
- Comprehensive documentation
- Installation guide
- Workflow examples
- Contributing guidelines

### Security
- JWT token authentication
- Secure credential storage via n8n
- HTTPS only API communication
### Documentation
- README with quick start guide
- INSTALLATION.md with detailed setup instructions
- EXAMPLES.md with practical workflow examples
- CONTRIBUTING.md for developers
